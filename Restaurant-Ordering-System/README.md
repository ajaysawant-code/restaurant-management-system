# 🍽️ Sawant Brother's Restaurant Ordering System

## 📖 Project Description
This is a Restaurant Ordering System developed using HTML, CSS, and JavaScript. Customers can browse the menu, add food items to the cart, generate a bill, and place orders. The system also includes an Admin Dashboard for managing orders.

## ✨ Features
- Restaurant logo and attractive UI
- Menu with food images
- Add items to cart
- Increase and decrease item quantity
- Remove items from cart
- Clear cart
- Bill generation
- Customer name and table number
- Unique Order ID
- Date and time of order
- Download bill as PDF
- Admin Dashboard
- Search order by Order ID
- Delete orders
- Total orders and total revenue
- Best-selling item statistics

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage
